# Focus & Calm Anchor

A unified interface designed to support deep work, task management, and nervous system regulation. This widget combines a timer, a persistent checklist, and a breathing pacer into a single "Glassmorphism" dashboard.

---

### ■ Objective
To provide a distraction-free digital environment that anchors the user's focus while offering tools to regulate stress levels during high-pressure work sessions.

---

### ■ Features

**◈ Focus Timer**
Customizable countdowns for deep work sessions (25m) and short breaks (5m), complete with audio cues upon completion.

**◈ Persistent Task Queue**
A checklist that saves progress locally to the browser. Refreshing the page does not result in data loss.

**◈ Visual Puzzle Logic**
A gamified visual layer where completing tasks progressively reveals a background image, providing a tangible sense of progress.

**◈ Resonance Breathing**
A visual pacer for Box Breathing (4-4-4-4) to assist in nervous system regulation and anxiety reduction.

### ■ Quick Start
1. **Download** the `index.html` file.
2. **Open** the file in any modern web browser (Chrome, Safari, Edge).
3. **Reset** the dashboard by navigating to the **Tasks** tab and clicking "Reset All Data".
